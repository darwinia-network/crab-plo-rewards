# Export CSV | Plo Rewards Export

- Fetch rewards data
- Export to csv file


## Local Serve

```bash
$ git clone git@github.com:darwinia-network/plo-rewards-export.git
# or git clone https://github.com/darwinia-network/plo-rewards-export.git
$ cd plo-rewards-export
$ yarn && yarn start
```
